## How to Add a New Week

Append a new row to the table above following this format:

```
| [Week #] | [Domain / Topic] | Q[start]-Q[end] | videos/sec-plus/week-[##]/clip.mp4 | videos/sec-plus/week-[##]/transcript.md |
```

Then:
1. Create the folder: `videos/sec-plus/week-[##]/`
2. Drop in `clip.mp4` (OpusClip export) and `transcript.md` (Fireflies export)
3. Log the addition in `CHANGELOG.md`
